monet — Multi-Agent Orchestration SDK

Pointer-only DAG orchestration with LangGraph + FastAPI + OpenTelemetry

Graph Topology

  • • Chat Graph — slash commands, triage, direct LLM
  • • Planning Graph — iterative plan + human approval gate
  • • Execution Graph — wave-based parallel execution via LangGraph Send API
  • • Kaizen hook — fires on all exits (success/partial/failed)

Agents & SDK

  • • Built-in: planner, researcher, writer, qa, analyst, publisher
  • • @agent decorator with typed context injection
  • • Signal system: plan_ready, needs_research, needs_human_review
  • • ArtifactStore for pointer-only state management
  • • CapabilityIndex for fleet-wide pool awareness

Deployment Scenarios

  • • S1: monet dev — local all-in-one (Aegra + Docker PG + Redis)
  • • S2: aegra serve + managed Postgres/Redis + monet worker
  • • S3: split fleet with N pools (fast, heavy, ...)
  • • S5: vendor-hosted orchestrator, customer-hosted workers
  • • S4: workers-only (InMemoryTaskQueue) for tests
